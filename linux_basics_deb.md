# Linux Basics
## Actions
Make Directory
```
mkdir (name)
```
Remove File
```
rm (filename)
rm -rF (directory name)
```
**Link files**
```
ln (source file) (where to put link)
```
**SSH (Secure Shell)**\
Use package openssh-client
```
ssh (user)@(addresS)
```
**SCP**\
Used to copy files via SSH connection
```
scp 
```
